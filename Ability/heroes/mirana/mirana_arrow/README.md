# 月神之箭

![](game/resource/flash3/images/spellicons/mjz_mirana_arrow.png)

射出一支绝对精准的箭矢，对第一个击中的敌人造成伤害。

Fires a long-range arrow with deadly precision, which damages the first enemy unit it strikes. 

**施法距离**：2000

**最远移动距离**：2067

**箭矢数量**：1（天赋 +2）

**箭矢碰撞范围**：115

**攻击力伤害**：60%/80%/100%/120%

**几率**：25%（天赋 +5%）

**伤害类型**：物理

**无视魔法免疫**：是
