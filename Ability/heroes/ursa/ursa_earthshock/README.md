# 震撼大地

![](game/resource/flash3/images/spellicons/mjz_ursa_earthshock.png)

熊战士向前跳跃`250`距离并猛击地面带来强烈的冲击，对附近范围内的所有敌方单位造成伤害并减速`4`秒。



**施法动作**：0.3+0

**范围**：385 (天赋 +600)

**伤害**：50/100/150/200

**额外力量伤害**：80%/100%/120%/140%

**移动速度减缓**：25%/35%/45%/55%

**减速时间**：4

**冷却时间**：10/9/8/7

**魔法消耗**：85

**伤害类型**：魔法

**无视魔法免疫**：否

**能否驱散**：是
