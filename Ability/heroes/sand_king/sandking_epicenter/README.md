# 地震

![](game/resource/flash3/images/spellicons/mjz_sandking_epicenter.png)

在2秒吟唱后，沙王向地中发送扰动，引起大地剧烈震动。所有范围内的敌人会受到伤害并被减速。每次后续震击都会提高伤害传播半径。

**施法动作**：0+0

**吟唱时间**：2

**震击范围**：300 - 2000

**震击次数**：6/8/10 (天赋 +10)

**震击伤害**：110

**额外力量伤害**：100%（天赋 +50%）

**移动速度减缓**：30%

**攻击速度降低**：60 

**减速时间**：3

**冷却时间**：100/90/80

**魔法消耗**：150/225/300

**伤害类型**：魔法

**无视魔法免疫**：否

**Debuff能否被驱散**：能
