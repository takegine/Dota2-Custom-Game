# 混乱之箭

![](game/resource/flash3/images/spellicons/mjz_chaos_knight_chaos_bolt.png)

对目标单位投掷一支神秘的能量箭。造成随机时间的眩晕和随机的伤害。



**施法动作**：0.4+0.73

**施法距离**：400/500/600/700

**作用范围**：325

**目标数量**：1 （天赋 +2）

**最低伤害**：60/90/120/150

**最高伤害**：180/220/260/300

**最短眩晕时间**：1/1/1/2

**最长眩晕时间**：2/3/4/5

**冷却时间**：10

**魔法消耗**：140

**伤害类型**：魔法

**无视魔法免疫**：否

**是否被驱散**：是

**林肯法球格挡**：是

**可以被躲避**：是
