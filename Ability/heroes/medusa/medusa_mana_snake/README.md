# 法力异蛇

![](game/resource/flash3/images/spellicons/mjz_medusa_mana_snake.png)



美杜莎每次攻击都将根据造成的伤害回复 14% 的魔法值。

Regenerates Medusa  mana equivalent to 14% of attack damage dealt.

*异蛇能够吸取敌人的法力。*

*The snake will absorb the enemy mana.*

**魔法偷取**：14%
