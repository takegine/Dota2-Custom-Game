# 米波王

![](game/resource/flash3/images/spellicons/mjz_meepo_meepo_king.png)



米波中的王者。如果敌人在光环范围内死亡，米波将获得额外经验值加成。

King of Meepo.If a unit dies while under this effect, Meepo gains bonus experience.

**攻击力加成**：10%/20%/30%

**攻击速度加成**：10/20/30

**护甲加成**：10/20/30

**魔法抗性加成**：10%/20%/30%

**生命恢复加成**：10/20/30

**经验获取加成**：10%/20%/30%

**经验获取范围**：1200

**模型大小变化**：10%/20%/30%
