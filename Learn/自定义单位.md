# 自定义单位

学习单位设置，可以查看 `npc_units.txt` 文件。

自定义单位模版：

```
	"npc_dota_creature_gnoll_assassin_2"
```

```json
"Creature"

			"AttachWearables"
			{
				"Wearable1" { "ItemDef"           "9327" }
```

如何获得饰品的ID？

1. 首先要找到所用物品或模型的名称，用Asset Browser 打开`models/heros` 和 `models/items` 查找
2. 对照 `items_game.txt` 文件，此文件在 `dota 2 beta\game\dota\pak01_dir.vpk` 里面，用解包软件导出
