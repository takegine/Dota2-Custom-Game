## 圣光吊坠



![](src/icon/mjz_great_holy_locket.png)

名称：圣光吊坠

效果：

​	**主动：充能**

​				使友方单位目标立刻根据能量点数回复相应的生命和魔法，每点能量恢复`90`点生命和魔法。每`3`秒获得`1`点能量（最多`20`点能量）。

​	**被动：神圣福佑**

​				自身提供的治疗效果提升`40%`。

**生命值加成**：300

**魔法值加成**：425

**生命恢复加成**：18

**全属性加成**：8



**合成**：圣洁吊坠(2500) + 恐鳌之戒(650) + 合成图纸(1550) =  4700
