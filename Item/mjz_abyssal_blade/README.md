## 深渊之刃 系列



![](src/icon/mjz_abyssal_blade.png)

名称：深渊之刃 系列

效果：

​	主动：**强击**

​			闪烁至敌方目标身旁并造成击晕，持续2秒。

​			无视技能免疫。  

​			**施法距离**：600 700 800 900 1000

​			**眩晕时间**：2

​	被动：**重击**

​	近战英雄的攻击有25%的概率击晕目标1.5秒并额外造成100点伤害。远程英雄的击晕概率为10%。

​	**触发几率(近战)**：25%

​	**触发几率(远程)**：10%

​	**伤害（魔法）**：100 110 120 130 150

​	**眩晕时间**：1.5

​	**冷却时间**：2.3



升级合成素材：秘银锤（1600） + 活力之球（1100）+ 卷轴（1750） = 4450
